# WordPress Plugin Development Skill

Build WordPress plugins with proper architecture, hooks, settings API, and security.

## Installation

```bash
npx degit indexsy/skills/wp-plugin-development ~/.clawdbot/skills/wp-plugin-development
```

## What's Included

- **SKILL.md** - Core plugin development methodology
- **references/structure.md** - Plugin file structure
- **references/lifecycle.md** - Activation, deactivation, uninstall
- **references/settings-api.md** - Settings pages and options
- **references/data-and-cron.md** - Data storage and scheduled tasks
- **references/security.md** - Nonces, capabilities, sanitization, escaping
- **references/debugging.md** - Debug techniques
- **scripts/detect_plugins.mjs** - Plugin detection script
