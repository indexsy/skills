# WordPress Block Development Skill

Build Gutenberg blocks with block.json, attributes, dynamic rendering, and proper tooling.

## Installation

```bash
npx degit indexsy/skills/wp-block-development ~/.clawdbot/skills/wp-block-development
```

## What's Included

- **SKILL.md** - Core block development methodology
- **references/block-json.md** - block.json metadata
- **references/registration.md** - register_block_type
- **references/attributes-and-serialization.md** - Attributes and save
- **references/dynamic-rendering.md** - render.php and render_callback
- **references/supports-and-wrappers.md** - Block supports
- **references/inner-blocks.md** - InnerBlocks usage
- **references/deprecations.md** - Block migrations
- **references/creating-new-blocks.md** - @wordpress/create-block
- **references/tooling-and-testing.md** - Build and test
- **references/debugging.md** - Debug techniques
