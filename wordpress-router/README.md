# WordPress Router Skill

Automatically route to the correct WordPress workflow based on project type (plugin, theme, block theme, Gutenberg, etc.).

## Installation

```bash
npx degit indexsy/skills/wordpress-router ~/.clawdbot/skills/wordpress-router
```

## What's Included

- **SKILL.md** - Core routing logic and decision tree
- **references/decision-tree.md** - Visual guide for project classification

## Usage

Point your AI agent to the SKILL.md file:

```
https://raw.githubusercontent.com/indexsy/skills/main/wordpress-router/SKILL.md
```
