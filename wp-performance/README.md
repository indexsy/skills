Optimize WordPress performance: profiling, database, caching, cron, HTTP API.

## Installation

```bash
npx degit indexsy/skills/wp-performance ~/.clawdbot/skills/wp-performance
```

- **references/wp-cli-profile.md** - WP-CLI profiling
- **references/wp-cli-doctor.md** - WP-CLI diagnostics
