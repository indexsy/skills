# WordPress PHPStan Skill

Configure and run PHPStan static analysis for WordPress projects.

## Installation

```bash
npx degit indexsy/skills/wp-phpstan ~/.clawdbot/skills/wp-phpstan
```

## What's Included

- **SKILL.md** - Core PHPStan methodology
- **references/configuration.md** - phpstan.neon setup
- **references/wordpress-annotations.md** - WP-specific typing
- **references/third-party-classes.md** - Handling plugin classes
