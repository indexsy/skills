# WordPress REST API Skill

Build custom REST API endpoints with proper controllers, schema, and authentication.

## Installation

```bash
npx degit indexsy/skills/wp-rest-api ~/.clawdbot/skills/wp-rest-api
```

## What's Included

- **SKILL.md** - Core REST API methodology
- **references/routes-and-endpoints.md** - register_rest_route
- **references/schema.md** - JSON Schema validation
- **references/authentication.md** - permission_callback
- **references/responses-and-fields.md** - Response shaping
- **references/custom-content-types.md** - CPT/taxonomy REST
- **references/discovery-and-params.md** - API discovery
