# WordPress Block Themes Skill

Build block themes with theme.json, templates, patterns, and style variations.

## Installation

```bash
npx degit indexsy/skills/wp-block-themes ~/.clawdbot/skills/wp-block-themes
```

## What's Included

- **SKILL.md** - Core block theme methodology
- **references/theme-json.md** - Global settings and styles
- **references/templates-and-parts.md** - HTML templates
- **references/patterns.md** - Block patterns
- **references/style-variations.md** - Theme variations
- **references/creating-new-block-theme.md** - Starting a new theme
- **references/debugging.md** - Debug techniques
