# WordPress Project Triage Skill

Inspect and classify WordPress projects to route to correct workflows.

## Installation

```bash
npx degit indexsy/skills/wp-project-triage ~/.clawdbot/skills/wp-project-triage
```

## What's Included

- **SKILL.md** - Core triage methodology
- **references/triage.schema.json** - JSON report schema
- **scripts/detect_wp_project.mjs** - Detection script
